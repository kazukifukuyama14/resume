# 職務経歴書

## 概要

クラウドインフラ構築・保守・運用専門としての職務経歴を公開しています。

## 職務経歴書サイト

**[GitHub Pages で職務経歴書を確認する](https://wan0ri.github.io/resume/)**

## プロジェクト構成

- **`docs/`**: 職務経歴書の詳細コンテンツ
- **`docs/index.md`**: 職務要約と自己 PR
- **`docs/README.md`**: 詳細職務経歴書

## 技術スタック

- **品質管理**: textlint, markdownlint
